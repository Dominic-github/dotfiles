# Alcritty

![alacritty](./alacritty.jpg)

## Install [_alacritty_](https://github.com/alacritty/alacritty):

```bash
sudo pacman -S alacritty
```

## Copy my config:

```bash
git clone https://github.com/Frey1a/dotfiles.git
cp -a dotfiles/.config/alacritty ~/.config
rm -rf dotfiles
```
