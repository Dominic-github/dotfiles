## Install [_ranger_](https://github.com/ranger/ranger)

```bash
sudo pacman -S ranger
```

## Copy my config:

```bash
git clone https://github.com/Dominic-github/dotfiles.git --depth 1
cp -a dotfiles/.config/ranger ~/.config
rm -rf dotfiles
```
