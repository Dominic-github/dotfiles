## 🚀 Install

> #### Copy and paste on terminal

###### with bash, zsh, ...

```bash
bash <(curl -s https://raw.githubusercontent.com/Frey1a/nvim/main/install.sh)
```

###### with fish

```bash
bash (curl -s https://raw.githubusercontent.com/Frey1a/nvim/main/install.sh | psub)
```

##### 🚩 Enjoy!
